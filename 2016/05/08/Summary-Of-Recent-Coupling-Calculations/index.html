Summary Of Recent $\left|H_{DA}\right|^2$ Calculations

Posted by Zheng (Zane) Ma || Photo By @candybud || on 2016-05-08
#NEq-MD #Electron Transfer #Theoretical Chemistry #Research Update

I. Current Status of IR-Control-ET Project

Regarding the Non-Equilibrium MD and associated coupling study, we have tried many types of calculations trying to explain/reproduce the significant IR-reduce-ET effect observed in experiments. Unfortunately currently we have not successfully reproduced the IR-reduce-ET effect. Listed in this document are the calculations we tried along with the detailed method and the observations we had from these calculations. As the project evolves, some of those old calculations may seem to be obsolete, but I still list them here for future reference so that this document could serve as an update and overview of the calculations we performed in this project.

II. List of Conducted Calculations

  1. Non-Equilibrium MD Run

    • Purpose: To reproduce the IR-reduce-ET effect from full MD propagation and ensemble-averaged coupling calculations.
    • Result Summary: No significant decrease on $\left<\left|H_{DA}\right|^2\right>$ was observed.
    • Computational Detail: The first step of this calculation is initial configuration sampling. In this step, we used classical sampling based on MM-optimized geometry. The IR-excited vibrational modes were calculated in MM and compared with DFT/B3LYP-calculated normal modes. Then the NEq-MD and Eq-MD trajectories were propagated using GAFF force field with dichloromethane (DCM) molecules as solvent in GROMACS. After taking snapshots, the electronic structure of each snapshot was calculated using INDO/S semi-empirical method followed by Block-Diagonalization calculation for the electronic couplings.
    • Discussion: This is the calculation where we established our NEq-MD methodology framework. However, because we did not observe very significant decrease on $\left<\left|H_{DA}\right|^2\right>$ as experiments reported, we started to examine all the computational methods in every step of our NEq-MD methodology and study their robustness. More importantly, according to the result, at $t=0$ moment, the non-equilibrium ensemble-averaged coupling ($\left<\left|H_{DA}\right|^2\right>_{NEq}$) is larger than the equilibrium ensemble-averaged coupling ($\left<\left|H_{DA}\right|^2\right>_{Eq}$). This is contrast to the experimental observations. In order to find the reason why we cannot achieve decreased coupling from NEq-MD calculations, we started to examine the robustness of our methodology.

  2. Mode Stretching Calculation With INDO/S:

    • Purpose: To Determine whether INDO/S method is correctly describing the electronic coupling between donor and acceptor under vibrational modulation of geometry.
    • Result Summary: INDO/S semi-empirical method significantly under-estimates the interaction between donor and acceptor orbitals and hence under-estimates coupling by approximately 1 order compared to DFT/B3LYP.
    • Computational Detail: In this set of calculations we stretched the GC-pair molecule geometry along MM-calculated normal mode coordinates. At each geometry, we calculate coupling using block-diagonalization based on both INDO/S-calculated and DFT/B3LYP-calculated electronic structure. Since it was found that INDO/S under-estimates the donor-acceptor coupling, we need to find another semi-empirical method which could correctly describe the coupling so that we can use it in NEq-MD run.

  3. Semi-Empirical Method Search:

    • Purpose: To find a semi-empirical method that gives donor-acceptor coupling that is on the same order as DFT/B3LYP method gives.
    • Result Summary: DFTB/3ob-3-1 gives $\left|H_{DA}\right|\sim 0.01eV$
    • Computational Detail: We tested several semi-empirical methods, including CNDO, PM3, PM6, AM1 and DFTB at MM-optimized geometry. Among all these methods, only DFTB/3ob-3-1 gives donor-acceptor coupling that is on the same order as DFT/B3LYP-calculated coupling. Here the coupling calculation method is block-diagonalization.

  4. Coupling at $t=0$ Using DFTB:

    • Purpose: To examine whether at $t=0$, DFTB/3ob-3-1 calculated coupling can reflect IR-reduce-ET effect.
    • Result Summary: Unfortunately there was no signficant difference between $\left<\left|H_{DA}\right|^2\right>_{NEq}$ and $\left<\left|H_{DA}\right|^2\right>_{Eq}$.
    • Computational Detail: Based on MM-optimized geometry, the GC-pair molecule initial configuration was sampled according to MM-calculated normal mode vectors. Without any MD propagation, we directly calculated the coupling using DFTB/3ob-3-1 and block-diagonalization at $t=0$.
    • Discussion: Since changing electronic structure calculation method to be DFTB/3ob-3-1 does not reflect IR-reduce-ET effect, we decided to use DFTB/3ob-3-1 optimized geometry and corresponding normal mode vectors instead of MM-calculated ones for initial configuration sampling.

  5. Coupling at $t=0$ Using DFTB-calculated Geometry and Normal Modes

    • Purpose: To examine at $t=0$, if DFTB/3ob-3-1 is used to optimize geometry and to calculate mode vectors, whether DFTB/3ob-3-1 calculated coupling can reflect IR-reduce-ET effect.
    • Result Summary: No IR-reduce-ET effect was observed.
    • Computational Detail: Based on DFTB/3ob-3-1 optimized geometry, the GC-pair molecule initial configuration was sampled according to DFTB/3ob-3-1 calculated normal mode vectors. Without any MD propagation, we directly calculated the coupling using DFTB/3ob-3-1 and block-diagonalization at $t=0$.
    • Discussion: In this set of calculations, more complex results were observed.
    • The most confusing one is that if all modes were involved in the configuration sampling process, both $\left<\left|H_{DA}\right|^2\right>_{NEq}$ and $\left<\left|H_{DA}\right|^2\right>_{Eq}$ are on the order of $\sim 1.0\times 10^{-4}eV$, which is consistent with experiment. However, if we only calculate the squared coupling at DFTB/3ob-3-1 optimized geometry, this value will be $\sim 1.0\times 10^{-5}eV$. This means the combination of stretching multiple modes (in sampling process, the vibrational level of non-IR modes is sampled based on Boltzmann distribution) results in strange geometries at which the calculated geometry is very large compared to optimized geometry.
    • In addition, the coupling calculated at optimized geometry indicates that DFTB/3ob-3-1 is not correctly describing the coupling either.
    • In this study we realized that on vibration groud state and low-excited states, the probability distributions of mass-weighted position provided by classical sampling are significantly different from quantum mechanical descriptions. As a result, we decided to use a Wigner-like probability distribution function for sampling the position and momenta for normal modes.
  6. Mode Stretching Calculation With DFT/B3LYP:
    • Purpose: To determine if electronic coupling has “asymmetric” dependency on any DFT/B3LYP-calculated vibrational mode.
    • Result Summary (on-going): No such vibrational mode between $1500\sim2000cm^{-1}$ frequency range and out-of-plane H-bond modes ($600\sim800cm^{-1}$) are not responsible for IR-reduce-ET effect either.
    • Computational Detail: In this set of calculations we stretched the GC-pair molecule geometry along DFT/B3LYP-calculated normal mode coordinates. At each geometry, we calculate coupling using both orbital-based GMH method and block-diagonalization based on DFT/B3LYP-calculated electronic structure. All electronic structure calculations were carried out in PCM model where dichloromethane is the solvent using Gaussian09.
    • Discussion: It is important to realize that the “asymmetric” behavior is the key to IR-control-ET, because “symmetric” dependency of coupling on vibrational modes will only cause the ensemble-averaged coupling to be approximately the same as the coupling calculated at optimized geometry. Hence during this set of mode stretching calculation, we have been trying to identify the modes that has “asymmetric” effect on calculated coupling. We also questioned ourselves whether block-diagonalization method is correct or not for this system, and the comparison between block-diagonalization calculated and GMH calculated couplings indicate that these two methods are consistent with each other. It was also found that including solvent model could significantly alter the coupling calculation result, so here we perform DFT/B3LYP calculations with the presence of PCM solvent model.

III. For Next Step

The mode stretching calculation with DFT/B3LYP is still not completed yet since there are hundreds of vibrational modes. But for the paper, I am writing them assuming no such mode could be found and it is probably more practical to make an argument that it is some other modes instead of H-bond modes that accepts vibrational energy through IVR process and subsequently affects the ET rate.
